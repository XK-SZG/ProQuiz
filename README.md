# ProQuiz

ProQuiz 是一款功能丰富、界面现代的纯前端刷题与考试系统。它提供了多种练习模式、详细的数据统计、错题本功能以及基于 MSSP 协议的安全成绩单校验，旨在为用户提供高效、沉浸式的学习体验。

## ✨ 核心特性

- **📊 学习数据总览 (Dashboard)**：实时统计平均得分率、累计刷题次数、错题库总数等，并提供近 10 次练习的趋势图表。
- **🎯 多样化练习模式**：支持顺序、随机、抽选等多种刷题方式，满足不同阶段的学习需求。
- **🧠 专属背题模式**：针对简答题设计的“四步记忆法”（提示 → 奇隐 → 偶隐 → 盲默），科学高效。
- **⏱️ 秒题训练**：根据题目类型和分值自动倒计时，超时自动判错，锻炼答题速度。
- **🎓 模拟考试**：全真模拟考试环境，自动计算总分。
- **❌ 错题本**：自动记录错题，方便集中复习和攻克薄弱环节。
- **🎨 个性化与暗黑模式**：支持系统级暗黑模式切换，内置 8 种炫酷的动态背景主题（如宇宙、波浪、代码雨等）。
- **🔒 MSSP 成绩单校验**：生成带有数字指纹和二维码的成绩单，支持教师端扫码核验，防止成绩篡改。

## 🚀 练习模式说明

1. **顺序双向**：列表展示，自由上下滚动答题。
2. **顺序单向**：逐题练习，答完即走，无法回退。
3. **随机双向**：打乱题库顺序，列表展示，自由滚动。
4. **随机单向**：打乱题库顺序，逐题练习，强化应变能力。
5. **双向随机抽选**：指定抽题数量，随机生成试卷，自由滚动。
6. **单向随机抽选**：指定抽题数量，随机生成试卷，逐题练习。
7. **背简答题**：专为简答题设计的记忆模式。
8. **秒题训练**：极限限时挑战。
9. **模拟考试**：标准考试模式。

## 🛠️ 技术栈

- **HTML5 / CSS3 / JavaScript (ES6+)**：纯原生前端实现，无需后端数据库。
- **Tailwind CSS**：用于快速构建现代化的响应式 UI。
- **Chart.js**：用于绘制学习趋势图表。
- **Lucide Icons**：精美的开源图标库。
- **QRCode.js**：用于生成成绩单核验二维码。

## 📦 安装与使用

1. 克隆或下载本项目到本地。
2. 使用任意现代浏览器（推荐 Chrome、Edge、Firefox）直接打开 `index.html` 即可运行。
3. **自定义题库**：编辑 `questions.js` 文件，按照文件内的格式要求替换或添加你自己的题目数据。

## ⚙️ 系统设置

在“设置”页面中，你可以：
- **题目呈现偏好**：开启/关闭选择题选项随机排列。
- **秒题模式设置**：自定义单选题、多选题和简答题的限时规则。
- **个性化外观**：切换动态背景主题并调整参数。
- **成绩单安全设置**：设置 **教师核验口令 (MSSP 协议密钥)**，用于成绩单的加密和解密，确保学生端和教师端口令一致才能成功核验。

### 🔐 密钥配置说明

为了保证 MSSP 协议密钥的安全性，系统默认将密钥配置移出了 Git 工作区。
1. 在 `js/` 目录下找到 `secret.example.js` 文件。
2. 将其复制并重命名为 `secret.js`。
3. 在 `secret.js` 中填入你的 `MSSP_INTERNAL_SECRET` 口令。
4. `secret.js` 已被 `.gitignore` 忽略，不会被提交到代码仓库中，请妥善保管。
